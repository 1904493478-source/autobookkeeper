# 快捷指令创建指南（PWA 版）

## 方法一：导入 .shortcut 文件（推荐）
1. 将 `自动记账.shortcut` 文件通过 AirDrop 发送到 iPhone
2. iPhone 会自动打开「快捷指令」App 并提示导入
3. 点击「添加快捷指令」完成

如果导入失败，请使用方法二手动创建。

---

## 方法二：手动创建（3 分钟）

### 步骤 1：新建快捷指令
打开「快捷指令」App → 右上角 + → 搜索添加以下操作：

### 步骤 2：按顺序添加 6 个操作

**① 截取屏幕**
- 搜索「截屏」
- 添加「截取屏幕」

**② 从图像中提取文本**
- 搜索「提取文本」
- 添加「从图像中提取文本」
- 语言选择：中文（简体）

**③ URL 编码**
- 搜索「URL 编码」
- 添加「URL 编码」
- 模式选择：编码为 URL

**④ 打开 URL**
- 搜索「打开 URL」
- 添加「打开 URL」
- 点击 URL 输入框，输入：
  `https://1904493478-source.github.io/autobookkeeper/?ocr=`
- 然后点击输入框，选择「选择魔法变量」→ 选择上一步「URL 编码后的文本」
- 最终 URL 格式为：`https://1904493478-source.github.io/autobookkeeper/?ocr=(编码后的OCR文本)`

**⑤ 显示通知**
- 搜索「通知」
- 添加「显示通知」
- 标题：`记账完成`
- 正文：`已记录一笔交易`

**⑥ 删除照片（可选）**
- 搜索「删除照片」
- 添加「删除照片」
- 自动删除临时截图（不执行则截图保留在相册）

### 步骤 3：命名 + 绑定
- 重命名为「自动记账」
- iPhone 设置 → 辅助功能 → 触控 → 轻点背面 → 轻点两下 → 选择「自动记账」

---

## 使用方法
1. 微信/支付宝支付后，停留在支付结果页面
2. 双击 iPhone 背面
3. 等待 1-2 秒（OCR 识别中）
4. 自动打开 PWA 并记账
5. 收到「记账完成」通知
